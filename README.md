# Cofrinho em Java

Este script é uma ferramenta simples para salvar e converter para reais valores de diferentes moedas, como o dólar e o euro.

Este é um projeto desenvolvido durante a disciplina Programação Orientada a Objetos, no curso Análise e Desenvolvimento de Sistemas do Centro Universitário Uninter, sob orientação do professor Leonardo Gomes. Ele utiliza conceitos como Classes, Herança e Polimorfismo para o estudo do Java.

## Java Piggy Bank

This script is a simple tool to save and convert the values of different currencies to brazilian reais, such as american dollars and euro.

This is a project developed during the Object-Oriented Programming course, in the Systems Analysis and Development program at Uninter University Center, under the guidance of Professor Leonardo Gomes. It uses concepts such as Classes, Inheritance and Polymorphism for the purpose of studying Java.
